# Дорожная карта: паритет с World Monitor через призму «страна ↔ Россия»

Цель: довести GEO PULSE до функционального класса [World Monitor](https://github.com/koala73/worldmonitor)
(56k★, AGPL-3.0), сохранив единственную ось анализа — отношения каждой страны мира с Россией.
Репозиторий открывается под AGPL-3.0, поэтому решения World Monitor можно заимствовать напрямую.

## Фаза 2 — фронтенд мирового контура (следующая)

Дашборд (отдельный репозиторий, ветка `nextjs`) поверх готового API v2:

1. **Карта мира** — choropleth по `GET /api/v2/map` (уровни ally→hostile), клик по стране → досье. Таймлапс-скраббер по `index_history`.
2. **Страница-досье страны** — `GET /api/v2/countries/{code}`: индекс + слагаемые (structural/media/boost), история, темы (`/topics`), заголовки (`/headlines`), сигналы, AI-брифинг (`/brief`).
3. **Лента сигналов** — `GET /api/v2/signals`, фильтры по типу/стране/severity, live-обновление.
4. **Мировой брифинг** — `GET /api/v2/brief` на главной.
5. **Health-панель** — `GET /api/v2/health`, бейдж «degraded coverage» на странах с упавшими тирами.

## Фаза 3 — данные и сущности

- Расширение `sources_world.yaml` до 100+ фидов (арабский, испанский, африканский пулы).
- `un-votes-loader` на все 99 стран (UN Digital Library) — уточнение структурного слоя.
- Реестр сущностей с alias-матчингом (паттерн World Monitor: alias 95% / keyword 70% / related 60%).
- Валютный слой: курсы к рублю/доллару, корреляция с медиа-тоном.

## Фаза 4 — продвинутая аналитика

- Геокодинг событий, конфликтные слои (UCDP-полы для RRI v2).
- Prediction markets. Кросс-региональные каскады (детектор `regional_cascade`).
- RRI v2: машинно-обучаемые веса по реакции индекса на реальные дипломатические события.

## Фаза 5 — слои worldmonitor (по исследованию docs/research/worldmonitor-layers.md)

Выполнено 2026-06-12: тир analytics (15 think-tank/OSINT-фидов с russia-фильтром),
флаги доверия источников (state_affiliated, propaganda_risk), страница /analytics,
торговля IMF IMTS на 97 стран, голосования ООН на 98 стран, цитаты в брифах,
языковая матрица покрытия. Дальше (каждый слой — отдельная спека):

- **Финансовый «радар изоляции РФ»** — композит 5–7 сигналов (RUB, MOEX,
  дисконт Urals–Brent, юань/рубль, SWIFT) с вердиктом по схеме WM Market Radar.
- **Энергетический слой** — CREA-счётчик импорта российских ископаемых по странам,
  TTF/price cap, curated-карта экспортных труб РФ, shadow-fleet события в сигналы.
- **Технологическая развязка** — экспортный контроль чипов, реэкспорт HS 8542
  через третьи страны (Comtrade), curated-реестр AI/tech-активов РФ.
- **Санкционный слой** — OFAC SDN diff + OpenSanctions, давление по программам/странам.
- **Prediction markets** — Polymarket-контракты вокруг РФ, медиа-опережение.
- **Инфраструктура деплоя** — сборка образов в GitHub Actions → registry,
  прод только тянет готовое (после OOM-инцидента 2026-06-12).

## Методологические данные реестра

`src/countries.py`: 99 стран, FIPS-коды для GDELT, членства (ОДКБ/ЕАЭС/СНГ/ШОС/БРИКС/НАТО/ЕС/G7),
флаги «недружественные»/санкции/война, курируемые поправки (`baseline_adj` с обоснованием в `baseline_note`).
Спорные позиции реестра (Армения↔ОДКБ заморожено, Сирия после смены власти, статус БРИКС-партнёров)
пересматриваются при обновлениях; история — в git.
