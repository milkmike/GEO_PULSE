## Что это?

GEO PULSE — открытая аналитическая платформа класса [World Monitor](https://github.com/koala73/worldmonitor), но с одной осью анализа: **как каждая страна мира относится к России**. Выбираешь страну — видишь индекс отношений, его структурные и медийные слагаемые, тематический расклад, сигналы и AI-досье.

Два контура покрытия:

| Контур | Стран | Как работает |
|--------|-------|--------------|
| **Tier 1 — глубокий** | 28 (СНГ-10 + 18 ключевых) | Собственные источники (RSS/Web/Telegram), LLM-анализ каждой статьи, температура −60°..+60°, сюжетные нити, аудиторный сплит |
| **Tier 2 — глобальный** | все 99 | GDELT: тон и объём освещения России национальными медиа каждой страны, ежедневно |

## Russia Relations Index (RRI v1)

Двухслойный индекс −100..+100 (методология вдохновлена CII v8 из World Monitor):

```
RRI = w_s × structural + w_m × media + boost     → полы/потолки от жёстких фактов

structural — медленный слой: блоки (ОДКБ/ЕАЭС/СНГ/ШОС/БРИКС vs НАТО/ЕС/G7),
             санкции, список «недружественных», голосования в ООН,
             курируемые поправки (стратегические договоры)
media      — быстрый слой: LLM-температура собственных СМИ (tier 1)
             или тон GDELT (tier 2)
boost      — события action level ≥ 5 за последние 14 дней (±15 max)

Веса: tier 1 — 0.45/0.55, tier 2 — 0.60/0.40 (медиа-сигнал tier 1 точнее)
Полы/потолки: война → cap −75 · санкции+недружественные → cap −5 ·
              Союзное государство → floor +25 · ОДКБ → floor −25

Уровни: ally ≥ +45 · partner ≥ +15 · neutral ≥ −15 ·
        cooling ≥ −40 · tension ≥ −70 · hostile < −70
```

## Сигнальная разведка медиаполя

| Сигнал | Что ловит |
|--------|-----------|
| `tier_convergence` | Событие освещают 3+ разных тира источников за 24ч — событие реально значимое |
| `official_silence` | Оппозиция/Telegram шумят, официальные СМИ молчат 6+ часов — сигнатура замалчивания |
| `velocity_spike` | Поток релевантных статей ≥2× от 30-дневной нормы — информационный шторм |
| `tone_shift` | Тон GDELT о России сместился на \|z\| ≥ 2 от 90-дневной нормы страны |
| `volume_surge` | Доля «российской» повестки в нац. медиа ≥2.5× нормы |
| `index_shift` | RRI сдвинулся на ≥10 пунктов за сутки или сменил уровень |

Анти-усталость: dedup-ключи + TTL на тип сигнала — одна аномалия не спамит.

## AI-брифинги

- **Мировой брифинг «Россия и мир»** — каждые 2 часа: главные сдвиги, эскалации, деэскалации, сигналы, что отслеживать. Вход хешируется — если состояние мира не изменилось, токены не тратятся.
- **Досье страны** — индекс + структурный фон + медиаполе + ключевые события + прогноз.

## Тематический расклад (16 тем)

Каждая статья получает 1–3 метки: дипломатия, военное, безопасность, экономика, энергетика, санкции, финансы, технологии, наука/космос, культура/спорт, миграция, информационное влияние, историческая память, организации, война в Украине (позиция), шпионаж.

## Tier 1: глубокое покрытие

Исходный контур проекта — 10 постсоветских стран (КЗ, АМ, УЗ, КГ, ТД, ТМ, АЗ, ГР, МД, БР) с 191 источником по 7 тирам (official, mainstream, independent, social, domestic_opposition, western_proxy, analytics) — расширен мировым каталогом ([sources_world.yaml](src/collectors/sources_world.yaml)): США, Великобритания, Германия, Франция, Италия, Польша, Украина, Турция, Китай, Индия, Иран, Израиль, Япония, Южная Корея, Бразилия, Сербия, Венгрия, Катар.

Ключевые находки СНГ-контура (медиа опережают дипломатию на 2–4 недели; аудиторный сплит двуязычных источников; «прозападный коридор» Молдова–Грузия) — в [исследовательских материалах](docs/).

## Быстрый старт

```bash
git clone <repo>
cd GEO_PULSE
cp .env.example .env   # OPENROUTER_API_KEY, JINA_API_KEY; опционально LLM_MODELS, OLLAMA_URL
docker compose up -d
```

Существующая база: примените миграцию `scripts/migrations/008_world_expansion.sql`.

**Встроенный мировой дашборд:** `http://localhost:8100/world` — интерактивная карта 99 стран, клик по стране открывает досье (индекс, слагаемые, темы, заголовки, сигналы, AI-брифинг).

Первичный прогон мирового контура:

```bash
docker compose run --rm gdelt-collector python scripts/collect_gdelt.py --days 90   # бэкфилл GDELT
docker compose run --rm ru-index python scripts/calc_ru_index.py                    # первый расчёт RRI
```
